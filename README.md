Features of the Dashboard:

Dashboard Overview
A summary section showing key metrics, review volume, and user rating trends.

Sentiment Analysis
Automatically classifies customer reviews as positive, neutral, or negative to understand overall user satisfaction.

Word Cloud
Visual representation of the most common words from customer feedback, highlighting frequent concerns and praises.

Linear Regression
A simple predictive model that analyzes the relationship between review ratings and other parameters.

Word-Search Review
A search feature that lets users filter reviews by specific keywords to identify themes, issues, or user sentiments.
